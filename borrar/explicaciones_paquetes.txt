EXPLICACIONES PARA PAQUETES DE GESTIÓN - TFG HYDROSYN
=====================================================

5.2.3.1. Gestión de Autenticación
---------------------------------
El paquete de Gestión de Autenticación es fundamental para garantizar la seguridad y el control de acceso al sistema Hydrosyn. Este módulo se encarga de verificar la identidad de los usuarios que intentan acceder a la plataforma, asegurando que solo el personal autorizado pueda operar el sistema de riego inteligente. 

La autenticación permite diferenciar entre distintos tipos de usuarios (propietarios, usuarios estándar) y asignarles los permisos correspondientes según su rol. Esto es crucial para mantener la integridad del sistema y proteger tanto los datos de operación como la configuración de los dispositivos físicos conectados.

Los casos de uso de este paquete abarcan desde el login inicial hasta el cierre de sesión, pasando por la gestión de credenciales y el control de accesos no autorizados.

5.2.3.2. Gestión de Usuarios
----------------------------
El paquete de Gestión de Usuarios permite administrar las cuentas de usuario que tienen acceso al sistema Hydrosyn. A través de este módulo, el propietario del sistema puede dar de alta nuevos usuarios, modificar sus datos personales, asignarles permisos específicos y gestionar su estado (activo/inactivo).

Esta gestión es esencial para sistemas donde múltiples personas necesitan acceder a la plataforma con diferentes niveles de responsabilidad. Por ejemplo, en una explotación agrícola pueden existir usuarios con permisos de solo lectura para consultar datos, mientras que otros pueden tener capacidades de configuración y control.

El sistema mantiene un registro detallado de todas las operaciones realizadas por cada usuario, facilitando la auditoría y el seguimiento de actividades dentro del sistema de riego inteligente.

5.2.3.3. Gestión de Sistemas
----------------------------
El paquete de Gestión de Sistemas permite administrar múltiples instalaciones o sistemas de riego dentro de la plataforma Hydrosyn. Cada sistema representa una configuración independiente con sus propios dispositivos, programaciones y parámetros de operación.

Este módulo es fundamental para empresas o usuarios que gestionan múltiples ubicaciones de riego, como diferentes invernaderos, campos de cultivo o zonas geográficas distintas. Permite centralizar la administración mientras mantiene la independencia operativa de cada sistema.

A través de este paquete se pueden crear nuevos sistemas, configurar sus parámetros generales, asociar dispositivos específicos y gestionar las relaciones entre diferentes instalaciones. Cada sistema mantiene su propia base de datos de sensores, actuadores y programaciones.

5.2.3.4. Gestión de ESP32
--------------------------
El paquete de Gestión de ESP32 se encarga de administrar los dispositivos microcontroladores que forman la base del sistema Hydrosyn. Los ESP32 son los nodos inteligentes que se conectan directamente con los sensores y actuadores del sistema de riego, actuando como intermediarios entre el mundo físico y la plataforma central.

Este módulo permite dar de alta nuevos dispositivos ESP32 en el sistema, configurar sus parámetros de comunicación, actualizar su firmware y monitorizar su estado operativo. Cada ESP32 puede gestionar múltiples sensores (humedad, temperatura, nivel de agua) y actuadores (bombas, válvulas, luces).

La gestión adecuada de estos dispositivos es crucial para garantizar la fiabilidad del sistema, ya que son responsables de ejecutar las programaciones de riego y recopilar datos en tiempo real de las condiciones ambientales y del sistema.

5.2.3.5. Gestión de Tanques
---------------------------
El paquete de Gestión de Tanques permite administrar los depósitos de agua que forman parte del sistema de riego Hydrosyn. Estos tanques pueden ser de diferentes capacidades y tipos (almacenamiento principal, tanques de reserva, depósitos de fertilizantes, etc.).

Este módulo facilita el control de los niveles de agua, la gestión de llenado y vaciado, y el mantenimiento de los tanques. Permite configurar alarmas por niveles bajos, programar llenados automáticos y llevar un registro histórico del consumo y los niveles.

La gestión eficiente de los tanques es fundamental para garantizar un suministro continuo de agua para el riego, optimizar el uso de recursos y prevenir situaciones de escasez que podrían afectar negativamente a los cultivos.

5.2.3.6. Gestión de Bombas
--------------------------
El paquete de Gestión de Bombas administra los dispositivos de bombeo que son el corazón del sistema de riego Hydrosyn. Estas bombas son responsables de mover el agua desde los tanques de almacenamiento hasta los diferentes puntos de riego.

Este módulo permite controlar la operación de las bombas, programar su funcionamiento, realizar mantenimientos preventivos y calibrar su rendimiento. Incluye funcionalidades para monitorizar el estado operativo, detectar fallos y optimizar el consumo energético.

La gestión adecuada de las bombas es esencial para garantizar un riego eficiente, prevenir daños por operación incorrecta y mantener la fiabilidad del sistema. Las bombas pueden operar manualmente o de forma automática según las programaciones establecidas.

5.2.3.7. Gestión de Luces
-------------------------
El paquete de Gestión de Luces permite administrar los sistemas de iluminación artificial que complementan el sistema de riego Hydrosyn. Estas luces pueden ser especialmente importantes en invernaderos o cultivos bajo techo donde se necesita controlar el fotoperíodo y la intensidad lumínica.

Este módulo facilita la programación de los ciclos de iluminación, el control de la intensidad, la gestión de diferentes tipos de luces (crecimiento, floración) y el mantenimiento histórico de su funcionamiento. Permite optimizar el consumo energético y coordinar la iluminación con los ciclos de riego.

La gestión inteligente de las luces contribuye a optimizar el crecimiento de las plantas, mejorar los rendimientos de los cultivos y reducir los costos operativos mediante programaciones automatizadas basadas en las necesidades específicas de cada tipo de cultivo.

5.2.3.8. Gestión de Registros
------------------------------
El paquete de Gestión de Registros se encarga de administrar toda la información histórica generada por el sistema Hydrosyn. Esto incluye datos de sensores, operaciones de riego, eventos del sistema, mantenimientos realizados y cualquier otra información relevante para el análisis y la toma de decisiones.

Este módulo permite consultar, analizar y gestionar grandes volúmenes de datos históricos, facilitando el seguimiento del rendimiento del sistema, la detección de tendencias y la optimización de las operaciones. Incluye herramientas para filtrado, exportación de datos y generación de informes.

La gestión adecuada de los registros es fundamental para el análisis predictivo, la auditoría de operaciones, el cumplimiento normativo y la mejora continua del sistema de riego inteligente. Estos datos también pueden utilizarse para entrenar modelos de machine learning que optimicen las futuras operaciones de riego.
